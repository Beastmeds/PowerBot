Befehle / Commands

Hier findest du eine Übersicht aller zur Zeit verfügbaren Bot-Befehle für PowerBot.

/help

Zeigt eine Hilfeübersicht zu verfügbaren Befehlen an.

/start

Startet die Kommunikation mit dem Bot und zeigt Einführungsinfos.

/join

Führt den Bot dazu, einer bestimmten Gruppe beizutreten (Admin-Erlaubnis erforderlich).

/leave

Lässt den Bot die Gruppe verlassen.

/ban

Sperrt einen Benutzer (nur Admins).

/unban

Hebt die Sperre für einen Benutzer auf (nur Admins).

/stats

Zeigt Statistiken zur Gruppennutzung und Bot-Aktivität an.

/broadcast

Sendet eine Nachricht an alle Gruppenmitglieder (nur Admins / Premium).

/welcome

Konfiguriert die Willkommensnachricht für neue Mitglieder.

/setprefix

Setzt das Befehls-Präfix (nur Admins).
